# ✍️ AI Landing Page Copywriter

An AI-powered tool that generates professional landing page copy instantly — Hero Headline, Subheadline, Features, and CTA — all in one click.

🌐 **Live Demo**: https://shivamsinghssr.github.io/ai-landing-copywriter/

---

## 🚀 Features

- Generate complete landing page copy using Google Gemini AI
- Single API call for all 4 sections (optimized for quota)
- Copy individual sections or export as TXT / PDF
- Clean dark UI built with React + Vite
- Fully deployed — Frontend on GitHub Pages, Backend on Render

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, Vite, CSS |
| Backend | FastAPI, Python |
| AI | Google Gemini API (gemini-2.5-flash-lite) |
| Deployment | GitHub Pages + Render |
| CI/CD | GitHub Actions |
